# Computer Security AES Project

## Overview

This repository contains a Computer Security project based on the implementation of the **Advanced Encryption Standard (AES-128)** algorithm using Python. The project demonstrates both encryption and decryption processes and explains how AES works as a symmetric key cryptographic algorithm.

The purpose of this project is to understand the internal structure of AES, including key expansion, substitution, permutation, mixing, and round-based encryption and decryption operations.

## About AES

The **Advanced Encryption Standard (AES)** is a widely used symmetric encryption algorithm. It uses the same secret key for both encryption and decryption. AES is commonly used for securing sensitive data in applications such as file encryption, secure communication, password protection, and network security.

AES-128 uses:

- 128-bit block size
- 128-bit key size
- 10 encryption rounds
- Substitution and permutation operations
- Symmetric key encryption
